# KineCalc-v2.4
An interactive Calculus and Kinematics visualizer built with Python and Pygame. It helps students visualize the First Principle of Differentiation by interactively shrinking $\Delta t$ to demonstrate how an Average Velocity (secant) converges into an Instantaneous Velocity (tangent).
💡**The Concept**
KineCalc demonstrates the core of Calculus: Limit. By interactively shrinking the time interval ($\Delta t$), users can observe how the Average Velocity (secant line) dynamically converges into the Instantaneous Velocity (tangent line) at any given point on a function.
🛠 **Key Features** 
🔷Real-time Equation Input: Visualize any mathematical function $f(t)$ using standard Python syntax.
🔷Dynamic Calculus Engine: Smooth, real-time visualization of secant-to-tangent transition.
🔷Live Data Analytics: Watch calculations update instantly for Avg. Velocity, Instantaneous Velocity, and Margin of Error.
🔷Cyberpunk Neon UI: A modern, distraction-free aesthetic designed for intense study sessions.
🔷Interactive Controls: Built-in zoom, pan, and real-time parameter sliders.

**How to Run:**
Ensure you have Python installed.
Install Pygame: pip install pygame
Run the script: python main.py

**Controls:**
TAB: View Technical Specifications/Credits.
Mouse Wheel: Zoom in/out of the graph.
Arrow Keys: Pan across the coordinate plane.
